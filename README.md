# Free Academic Website

This is a no-cost static website for a biostatistician / PhD candidate profile.

## What You Get

- No hosting bill when published with GitHub Pages
- No monthly cost
- No build step
- No paid domain required
- Works with the free `yourusername.github.io` address

## Edit Your Information

- biography wording

Double-click `index.html` to open it in your browser.

## Publish For Free On GitHub Pages

1. Create a free GitHub account if you do not already have one.
2. Create a new public repository named exactly `yourusername.github.io`.
3. Upload everything inside this `academic-website` folder.
4. Go to the repository's `Settings` > `Pages`.
5. Choose the `main` branch and `/root` folder.
6. Your site will be available at `https://yourusername.github.io`.

Do not buy a custom domain if you want the total cost to stay at `$0`.
